Example Usage:

- To create a new vault file

```bash
$> vault init .env.vault
```

- To set a new key-value pair

```bash
$> vault key-set .env.vault
```

- To list all key-value pairs
```bash
$> vault key-list .env.vault
```

- To remove a key-value pair
```bash
$> vault key-remove .env.vault
```
